INTRODUCTION
------------

The PDF Export module gives the ability to export as PDF any kind of content.
Basically you are able to select the content to be include or exclude using
jQuery selector.

REQUIREMENTS
------------

This module requires the following modules:

 * Views (https://drupal.org/project/views)
 * Libraries (https://www.drupal.org/project/libraries)


INSTALLATION
------------

1. Download the library (http://www.mpdfonline.com/repos/MPDF_5_7.zip) from the
   MPDF site (http://www.mpdf1.com/mpdf/index.php)

2. Rename folder to "mpdf" inside sites/all/libraries

3. Go to admin/reports/status and check if the mpdf line is green
